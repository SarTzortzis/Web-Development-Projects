Κατασκευή Ιστοσελίδων

Η ανάπτυξη εφαρμογών Ιστού είναι το έργο που εμπλέκεται στην ανάπτυξη μιας ιστοσελίδας για το Διαδίκτυο (World Wide Web) ή ενός ενδοδικτύου (ένα ιδιωτικό δίκτυο). Η ανάπτυξη ιστού μπορεί να κυμαίνεται από την ανάπτυξη μιας απλής στατικής σελίδας απλού κειμένου έως σύνθετων εφαρμογών ιστού, ηλεκτρονικών επιχειρήσεων και υπηρεσιών κοινωνικού δικτύου. Μια πιο περιεκτική λίστα εργασιών στις οποίες αναφέρεται συνήθως η ανάπτυξη Ιστού, μπορεί να περιλαμβάνει τη σχεδίαση Ιστού, τη σχεδίαση Ιστού, την ανάπτυξη περιεχομένου Ιστού, τη σύνδεση πελάτη, τη δέσμη ενεργειών από πλευράς πελάτη / διακομιστή, τη διαμόρφωση διακομιστή Ιστού και δικτύου και την ανάπτυξη ηλεκτρονικού εμπορίου.

Μεταξύ των επαγγελματιών του Διαδικτύου, η "Ανάπτυξη Ιστού" αναφέρεται συνήθως στις κύριες μη σχεδιαστικές πτυχές της δημιουργίας ιστοσελίδων: γράφοντας σήμανση και κωδικοποίηση. Η ανάπτυξη ιστού ενδέχεται να χρησιμοποιεί συστήματα διαχείρισης περιεχομένου (CMS) για να κάνει τις αλλαγές περιεχομένου ευκολότερες και διαθέσιμες με βασικές τεχνικές δεξιότητες. Για μεγαλύτερους οργανισμούς και επιχειρήσεις, οι ομάδες ανάπτυξης Ιστού μπορούν να αποτελούνται από εκατοντάδες άτομα (προγραμματιστές Ιστού) και να ακολουθούν τυπικές μεθόδους όπως οι μεθοδολογίες Agile κατά την ανάπτυξη ιστοσελίδων. Οι μικρότεροι οργανισμοί ενδέχεται να απαιτούν μόνο έναν μόνιμο ή συμβαλλόμενο προγραμματιστή ή δευτερεύουσα ανάθεση σε σχετικές θέσεις εργασίας, όπως ένας γραφίστας ή ένας τεχνικός πληροφοριακών συστημάτων. Η ανάπτυξη Ιστού μπορεί να είναι μια συνεργατική προσπάθεια μεταξύ τμημάτων και όχι του τομέα ενός καθορισμένου τμήματος. Υπάρχουν τρία είδη εξειδίκευσης προγραμματιστών Ιστού: προγραμματιστής front-end, προγραμματιστής back-end και προγραμματιστής πλήρους στοίβας. Οι προγραμματιστές front-end είναι υπεύθυνοι για τη συμπεριφορά και τα γραφικά που εκτελούνται στο πρόγραμμα περιήγησης χρήστη, ενώ οι προγραμματιστές back-end ασχολούνται με τους διακομιστές.

HTML

Η HTML (Hyper Text Markup Language) είναι η τυπική γλώσσα σήμανσης για έγγραφα που έχουν σχεδιαστεί για εμφάνιση σε πρόγραμμα περιήγησης ιστού. Μπορει να συνδιαστεί με τεχνολογίες όπως Cascading Style Sheets (CSS) και γλώσσες scripting όπως η Javascript

CSS

H CSS (Cascading Style Sheets) είναι μια γλώσσα φύλλου στυλ που χρησιμοποιείται για την περιγραφή της παρουσίασης ενός εγγράφου γραμμένου σε γλώσσα HTML. Η CSS έχει σχεδιαστεί για να επιτρέπει τον διαχωρισμό της παρουσίασης του περιεχομένου, συμπεριλαμβάνομένης της διάταξης, των χρωμάτων και των γραμματοσειρών.

Javascript

Παράλληλα με την HTML και την CSS, η Javascript είναι μια απο τις βασικές τεχνολογίες του παγκόσμιου ιστού. Πάνω απο το 97% των ιστότοπων την χρησιμοποιούν απο την πλευρά του πελάτη την συμπεριφορά της ιστοσελίδας ενώ συχνά ενσωματώνονται και εξωτερικές βιβλιοθήκες.
