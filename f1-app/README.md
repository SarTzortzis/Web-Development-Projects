# Formula 1 Web Application
✔️ Fully responsive <br/>
✔️ React <br/>
✔️ MUI <br/>
✔️ API <br/>
## F1 standings is a web application that uses <b>Ergast Api</b> (http://ergast.com/mrd/). <br/>
